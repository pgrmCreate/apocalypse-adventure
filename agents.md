# Instructions agent

- La constante `CACHE_BUSTER` dans `game-constants.js` doit etre incrementee a chaque nouvelle version ou tache afin de forcer le rafraichissement du cache.
- L'univers du jeu est un survival urbain brutal : pas d'actions fantaisistes, chaque scene doit rester crue, logique et montrer le poids des choix (blessures, ressources manquantes, morale qui casse).
