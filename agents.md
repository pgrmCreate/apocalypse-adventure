# Instructions agent

- La constante `CACHE_BUSTER` dans `game-constants.js` doit etre incrementee a chaque nouvelle version ou tache afin de forcer le rafraichissement du cache.
- L'univers du jeu est un survival urbain brutal : pas d'actions fantaisistes, chaque scene doit rester crue, logique et montrer le poids des choix (blessures, ressources manquantes, morale qui casse).
- Le combat repose sur des fenetres courtes de pre-contact (encore plus serre pour le couteau), une esquive contextuelle juste avant l'impact et la possibilite de repousser un adversaire tres proche.
- Le loot aleatoire ne tombe plus automatiquement : les objets minimum vont directement dans l'inventaire, le reste se decouvre via l'action de fouille progressive (durée propre à chaque lieu, progression interruptible avec items qui apparaissent peu à peu).
